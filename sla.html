Introduction to Second Language Acquisition

Second Language Acquisition (SLA) is the study of how people learn a language subsequent to their native (first) language. It investigates the processes, factors, and outcomes of learning additional languages.

Core Terminology

L1 vs L2

Mother Tongue (L1): First language acquired naturally in infancy

Second/Foreign Language (L2): Any language learned after L1

  • Second Language: Learned where widely spoken (immersion)
  • Foreign Language: Learned outside native-speaking context

Input vs Output

Input (Krashen): Language data learner hears/reads

  • Must be comprehensible (i+1)
  • Massive exposure needed
  • Relevant to learner's needs

Output (Swain): Language learner produces

  • Noticing function
  • Hypothesis testing
Foundational SLA Concepts

Interlanguage

Definition: Unique linguistic system developed by L2 learners, intermediate between L1 and L2 (Selinker, 1972)

Key Features:

  • Systematic: Rule-governed, not random
  • Permeable: Open to new input and change
  • Variable: Changes with context
  • Dynamic: Constantly evolving

Significance: Moved focus from error analysis to understanding learner's internal system

Systematic Dynamic Rule-governed

Transfer

Definition: Influence of L1 structures on L2 learning

Positive Transfer

L1 structure aligns with L2, facilitating learning

Example: Cognates between similar languages

Negative Transfer (Interference)

L1 structure conflicts with L2, causing errors

Example: Word order errors, phonological interference

Fossilization

Definition: Cessation of interlanguage development before reaching target language norms (Selinker, 1972)

Characteristics:

  • Persistent errors despite exposure/instruction
  • Common in phonology and morphology
  • May relate to neural commitment or psychological factors

Debated Aspects: Permanency, universality, causes

Plateau Persistent Errors Ceased Development

Error vs Mistake

Error

Competence issue

  • Systematic deviation
  • Reveals knowledge gaps
  • Resistant to self-correction
  • Shows interlanguage stage

Mistake

Performance issue

  • Random, non-systematic
  • Due to fatigue, inattention
  • Can self-correct when prompted
  • Doesn't reflect competence level

Language Competence Models

Linguistic Competence (Chomsky)

Underlying, unconscious knowledge of language system (grammar, syntax, morphology)

Linguistic Performance

Actual use of language in concrete situations, affected by nonlinguistic factors (fatigue, memory)

Communicative Competence (Hymes, 1972)

Knowing when, where, and how to use language appropriately in social contexts (sociolinguistic, discourse, strategic competence)

Acquisition vs Learning

"Acquisition is the way people develop ability in a first language... Learning is the process that results in knowing about language." - Stephen Krashen

Aspect Acquisition Learning
Process Subconscious, implicit Conscious, explicit
Similarity Like L1 acquisition Formal instruction
Focus Meaning of communication Form of language
Result Spontaneous, fluent use Knowledge about language
Function Primary language ability "Monitor" for editing output
Conditions Meaningful interaction Time + focus on form + rule knowledge

Krashen's Monitor Model

Acquisition-Learning Hypothesis: Distinct processes for developing language ability

Monitor Hypothesis: Learned knowledge acts as editor for acquired language

Conditions for Monitor Use: Time, focus on form, knowledge of the rule

Rationalism in SLA

Rationalism is a philosophical tradition privileging reason as the primary source of knowledge, associated with Descartes, Spinoza, and Leibniz. It fundamentally challenges empiricist claims that all knowledge derives from sensory experience.

Core Concepts of Rationalism

Innate Ideas

Knowledge exists prior to experience, hardwired into human cognitive architecture

Prior Knowledge

Certain truths known independently of sensory input through pure reason

Deductive Reasoning

Truth derived through logical inference from self-evident axioms

Mind-World Correspondence

Rational structure of mind mirrors rational structure of reality

Rationalist View of Language Acquisition

Key Idea: Language acquisition is not learning but linguistic maturation/growth

Role of Input: Triggers parameter setting and provides lexicon evidence

Syntax Principles: Never learned - already specified by Universal Grammar

Stage Description Rationalist Interpretation
Initial State Learner begins with innate capacity Universal Grammar: innate principles + unset parameters
Input Triggers Exposure to language data Input triggers parameter setting, not data to be learned
Parameter Setting Language-specific rules acquired Language faculty fixes parameters based on linguistic data
Steady State Mature linguistic competence Internalized grammar of target language emerges
SLA Questions from Rationalist Perspective
  1. Do adult L2 learners retain access to Universal Grammar?
  2. Can parameters be reset, or do L1 settings constrain L2 development?
  3. Does adult L2 acquisition rely more on general cognitive mechanisms?

Research positions vary: Some argue for continued UG access (Flynn & Martohardjono, 1994), others propose greater reliance on cognitive mechanisms in adults.

Empiricist Critiques

  • Poverty of stimulus argument challenged: Input may be richer than claimed (Pullum & Scholz, 2002)
  • Corpus studies show child-directed speech contains informative grammatical patterns
  • If sufficient positive evidence exists, need for innate knowledge diminishes

Usage-Based Critiques

  • Alternative explanations without innate knowledge (Tomasello, 2003)
  • Development proceeds from concrete to abstract patterns gradually
  • Processes of generalization and categorization explain acquisition

Additional Criticisms

  • Cross-Linguistic Variation: Languages show more diversity than principles-and-parameters predicts
  • Individual Variation: Children show different acquisition rates/paths, suggesting learning rather than uniform biological maturation

Major Theories of Language Acquisition

Behaviorist Theory

Theorist: B.F. Skinner (1957)

Core Idea: Language acquisition as habit formation through operant conditioning

Mechanisms:

  • Imitation of adult speech
  • Reinforcement of correct forms
  • Stimulus-response associations
  • Environmental shaping

Limitations: Cannot explain creative language use beyond learned patterns

Imitation Reinforcement Habit Formation

Nativist Theory

Theorist: Noam Chomsky (1965)

Core Idea: Humans possess innate linguistic knowledge (Universal Grammar)

Key Concepts:

  • Language Acquisition Device (LAD)
  • Universal principles and parameters
  • Poverty of the stimulus argument
  • Innate biological predisposition

Strength: Explains rapid acquisition despite limited input

Innate Capacity UG LAD

Interactionist Theory

Theorists: Jerome Bruner (1983), Lev Vygotsky (1978)

Core Idea: Language emerges through social interaction and cognitive development

Key Concepts:

  • Scaffolding by proficient speakers
  • Zone of Proximal Development
  • Communicative functions drive development
  • Social context crucial

Emphasis: Dynamic interplay of cognitive and social factors

Scaffolding ZPD Social Interaction
Current Understanding

Integration of Perspectives: Current research suggests acquisition involves combination of mechanisms:

  • Innate predispositions constrain possible grammars
  • Statistical learning identifies patterns in input
  • Social motivation drives communicative development
  • Multiple cognitive systems integrate in acquisition

Signs of Progress in SLA

Interlanguage Development: L2 learners progress through predictable stages with characteristic errors that signal cognitive growth. Mistakes are evidence of evolving linguistic systems.

Selinker's Interlanguage Theory (1972): Five central processes shaping interlanguage:

  1. Language Transfer: Influence from L1 structures
  2. Transfer of Training: Impact of instructional methods
  3. Learning Strategies: Conscious approaches to acquisition
  4. Communication Strategies: Techniques for maintaining interaction
  5. Overgeneralization: Extending rules beyond appropriate contexts
Six Key Signs of L2 Development
Sign Description Example What It Indicates
1. Overgeneralization Applying learned rules too broadly "good" instead of "went" Rule acquisition, active learning
2. U-Shaped Development Temporary regression after correct usage Using "came" then "comed" Mental grammar restructuring
3. Increased Complexity Using sophisticated structures with errors Complex sentences with mistakes Risk-taking, system expansion
4. Self-Correction Monitoring and fixing own errors Correcting grammar during speech Metalinguistic awareness development
5. Variability Reduction More consistent performance Same accuracy across contexts Automatization, consolidation
6. Productive Use Generating novel utterances New sentences not memorized Internalized system building
Practice: Analyzing Developmental Errors

Sample EFL Learner Errors for Analysis:

  1. "I good to the market yesterday." (Overgeneralization of -ed past tense)
  2. "My brother is more taller than me." (Double comparative)
  3. "Although I was tired, but I finished my homework." (L1 transfer - conjunction redundancy)
  4. Consistent use of "I have seen" instead of mixing "I saw/I have seen" (Avoidance of simple past)
  5. "I have... I had a good time at the party." (Self-correction attempt)
  6. "He comed home" (after previously using "came" correctly) (U-shaped development)

Discussion Question: How can recognizing developmental signs change our perspective on learner errors?

First Language Acquisition Stages

First language acquisition progresses through identifiable developmental stages, showing systematic properties of language development.

Pre-linguistic Stage
0-12 months
  • Crying: Initial vocalizations
  • Cooing (2 months): Soft, vowel-like sounds
  • Canonical babbling (6-10 months): Reduplicated syllables ("bababa")
  • Perceptual narrowing: Attuning to native language phonology
  • Categorical perception: Discerning phonetic contrasts
One-Word Stage (Holophrastic)
12-18 months
  • First true words: Reference familiar objects, people, actions
  • Holophrastic speech: Single word conveys complete idea ("milk" = "I want milk")
  • Comprehension > Production: Understand more than can produce
  • Vocabulary: ~50 words by 18 months
Two-Word Stage (Telegraphic)
18-24 months
  • Word combinations: Express semantic relationships
  • Telegraphic speech: "Daddy go", "eat cookie", "Mommy shoe"
  • Emergent syntax: Understanding without function words/inflections
  • Vocabulary explosion: Rapid expansion
Multi-Word Stage
24-30 months
  • Expanded sentences: Beyond two words
  • Inclusion of function words: Articles, prepositions, conjunctions
  • Inflectional morphology: Verb tenses, plurals
  • Overgeneralization: "good" (went), "mouses" (mice) - shows rule learning
  • Active rule-governed learning: Not mere imitation

Stage Age Key Developments Theoretical Connections

  • Mastery of grammatical morphemes (Brown, 1973)

  • Narrative abilities with temporal sequencing

Second Language Acquisition Factors

Age Factors

Critical Period Hypothesis: Optimal period for language acquisition

Younger Learners: Often better pronunciation, intuitive grammar

Older Learners: Better at explicit learning, metalinguistic awareness

Age-Related Questions for Rationalism:

  • Does age-related decline support UG-based approaches?
  • Is there continued UG access in adulthood?
  • How does CPH relate to parameter setting?

Cognitive Factors

Aptitude: Innate ability for language learning

Learning Strategies: Conscious approaches to acquisition

Metalinguistic Awareness: Ability to think about language

Working Memory: Crucial for processing linguistic input

Automatization: Developing automatic language use

Affective Factors

Motivation: Integrative vs. instrumental

Anxiety: Can facilitate or hinder learning

Self-confidence: Belief in ability to learn

Attitude: Toward target language and culture

Affective Filter (Krashen): Emotional barrier to acquisition

Social Factors

Input Quality: Comprehensible, abundant, relevant

Interaction Opportunities: Meaningful communication

Social Identity: Relationship with target language community

Power Dynamics: Social positioning affecting acquisition

Vygotskyan Perspective: Learning through social interaction
